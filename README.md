# 差旅请假记录

玻璃拟态风格的全年度差旅 / 请假记录单页应用。纯前端、单文件、零构建。

## 线上地址
🌐 https://smsc123.github.io/travel-leave/

## 功能
- 全年 12 月日历总览，点击月份进入二级详情页
- 多日出差 / 请假（日期区间）
- 玻璃拟态自定义日期区间选择器
- 日历标记：外圈圆环（差旅/请假）+ 内部圆点（已报销/未报销）
- 汇总统计、搜索、状态筛选
- 云端同步：URL 链接（含数据）/ JSON 导入导出
- URL 路由直达：`?y=年&m=月&d=日期` 打开即定位

## 技术
原生 HTML + 内联 CSS + 原生 JS，仅 CDN 引入 `lz-string`。数据存浏览器 `localStorage`。

## 文件
- `index.html` —— 应用本体（单文件）
- `项目文档_差旅请假记录.md` —— 结构 / 视觉规范 / 续改指引

## 部署
GitHub Pages，从 `gh-pages` 分支根目录读取。源码在 `main` 分支。
